# 💻 Desafios de Lógica de Programação - DIO

Este repositório consolida todos os meus projetos e desafios práticos desenvolvidos durante as aulas de Lógica de Programação da [Digital Innovation One (DIO)](https://www.dio.me/).

## 📂 Estrutura dos Projetos

O repositório está dividido nos seguintes desafios:

1. **[Desafio do Felipão](./desafio%20do%20felipao)**
   - Lógica inicial de programação.
2. **[Segundo Desafio Felipão](./segundo%20desafio%20felipao)**
   - Desafio prático utilizando JavaScript para cálculo de partidas rankeadas.
3. **[Terceiro Desafio Felipão](./terceiro%20desafio%20felipao)**
   - Escrevendo e manipulando as classes de um jogo.

## 🛠️ Tecnologias Utilizadas
- **JavaScript**
- **Lógica de Programação**
- **Git e GitHub**
